    Windows 10 IoT Core Development for the Raspberry Pi 2 (Lee Perkins)

    If the thought of being able to easily write C# applications to run on a computer the size of a credit card sounds awesome then Lee's talk on Windows 10 IoT on the Raspberry Pi should prove to be interesting. Lee started his love of IoT with Arduino microcontrollers but quickly fell in love with the Raspberry Pi. The Raspberry Pi ("Pi") is a credit card sized computer with USB, HDMI, audio and General Purpose I/O (GPIO) pins to allow practically endless hardware enhancement opportunities. From the beginning it has supported various distributions of the Linux operating system, but Microsoft announced last year that they were committed to releasing a version of Windows 10 called Windows 10 IoT Core that would run on the Pi for the maker community. This is an "embedded systems" version of Windows. You won't find a desktop or a start menu here, but you will find all the OS needed for your C# applications (GUI or not) to run. In Lee's talk he will walk through how to install Windows 10 IoT on the Pi, set up the development environment in Windows 10 on a PC, write software on the PC that will run on the Pi, and debug that software running on the Pi with Visual Studio on the PC.

    Lee Perkins is a senior software engineer for IssueTrak. He has been involved in the software field for over 30 years. He started as a hobbyist programmer in the ‘80s and started his professional career as a software developer in 1990 at a company that no longer exists, Office Warehouse. His passion for programming has fueled his ever continuing desire to learn and advance in the field. He accomplished one of his dreams from 2005 through 2011 by running his own contract software development business. In the beginning of 2012 he became a full time employee of IssueTrak where he had been contracting for over 6 years. His more recent experiences include ASP.NET MVC, C#, JavaScript, and IoT. His passion for IoT combined with his decades of experience with Microsoft have led him to a fascination with Windows 10 IoT on the Raspberry Pi.

    Azure App Services (Sidney Andrews)

    While you have seen demos showing how you can deploy a web application to Azure, the Azure Web Apps service can do a lot more. Did you know that you can run Node, Bash or Python jobs, create a local Git repo, use Monaco to edit code in-browser, and override Web.config settings at run-time? That list just covers Web Apps, effectively 25% of the new Azure App Service. In this talk we will cover the whole service and show you how you can: extend your application's logic (ex. adding SMS support) through a minimal-code workflow solution, add Swagger-driven APIs that can be used by other applications, and even add a service and SQL solution that creates database tables to match the schema of your JSON.

    Sidney Andrews is a Microsoft Certified Trainer and Azure consultant with a background in ASP.NET and XAML development. Sidney has driven efforts to develop and deliver Azure readiness training through channels such as the Ignite Conference, Azure University Roadshow, Microsoft Virtual Academy, Microsoft Official Courseware, internal Microsoft training and even public whitepapers. Sidney has a series of FREE videos on YouTube if you want to learn more about Azure services from a developer perspective.
